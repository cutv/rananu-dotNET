Unlock file AutoITX3 (nhấp phải chọn Properties > Unlock > Ok)
Theo các step này:
After download completes, Copy & paste it in c:\autoit\PASTE HERE (c:\autoit\AutoItX3.dll)
Open Command Prompt as an Adminstrator
In Command Prompt  > cd c:\windows\systems32 [Press Enter]
c:\windows\systems32 > regsvr32 c:\autoit\AutoItX3.dll [Press Enter]
Prompt will appear with message “registration Succeeded”
